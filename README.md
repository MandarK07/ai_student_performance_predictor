# 🎓 AI-Based Student Academic Performance Predictor

This project predicts student academic performance using machine learning. It helps educators identify at-risk students early and make data-driven decisions.

## 🔧 Tech Stack
- Backend: FastAPI, Python, scikit-learn, XGBoost
- Frontend: React, Chart.js
- Deployment: Docker, GitHub Actions

## 🚀 Getting Started
1. Clone the repo
2. Set up Python environment
3. Install dependencies
4. Run backend and frontend

## 📁 Folder Structure
- `src/`: Core backend logic
- `frontend/`: React dashboard
- `data/`: Raw and processed datasets
- `models/`: Saved ML models
